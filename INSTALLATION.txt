# AI Podcast Generator - Installation & Setup

## Plugin Overview

This WordPress plugin generates AI-powered podcasts from your articles using:
- **OpenRouter API** for script generation (supports Claude, GPT-4, etc.)
- **OpenAI TTS** for audio generation (11 different voices)
- **Tavily Search** for web research (optional)
- **Action Scheduler** for background processing

## Installation Steps

### 1. Install Dependencies

```bash
cd wp-content/plugins/ai-podcast-generator
composer install
```

This will install Action Scheduler automatically.

### 2. Upload to WordPress

- Upload the entire `ai-podcast-generator` folder to `/wp-content/plugins/`
- Activate the plugin through the 'Plugins' menu in WordPress

### 3. Configure API Keys

Go to **AI Podcasts > Settings** and add your API keys:

**OpenRouter API Key** (Required)
- Sign up at: https://openrouter.ai/
- Get API key: https://openrouter.ai/keys
- Recommended model: anthropic/claude-sonnet-4

**OpenAI API Key** (Required)
- Sign up at: https://platform.openai.com/
- Get API key: https://platform.openai.com/api-keys
- Used for TTS (text-to-speech)

**Tavily API Key** (Optional)
- Sign up at: https://tavily.com/
- Enhances podcasts with latest web research
- Can be skipped if you disable web search

## Features

### Manual Generation
1. Go to **AI Podcasts > Generate**
2. Select a source article
3. Choose duration (5, 10, 15, 20, 30 minutes)
4. Select language (9 languages supported)
5. Choose number of hosts (1-3)
6. Optionally add a guest expert
7. Customize voice mapping
8. Add custom intro/outro
9. Click "Generate Podcast"

### Automated Generation
1. Go to **AI Podcasts > Settings**
2. Enable "Auto-Generation"
3. Set frequency (every 15min, 30min, hourly, etc.)
4. Configure default settings
5. Plugin will automatically:
   - Review recent articles
   - Select the best one using AI
   - Generate podcast in background

## How It Works

### Background Processing
- Uses Action Scheduler for reliable background tasks
- Avoids WordPress timeout issues
- Processes can take 5-15 minutes depending on length
- Check dashboard for progress

### Script Generation
1. AI reads your article
2. Optionally searches web for latest info (Tavily)
3. Creates conversational podcast script
4. Multiple hosts discuss the topic naturally

### Audio Generation
1. Script is chunked (OpenAI has 4096 char limit per request)
2. Each speaker gets appropriate voice
3. Audio chunks are generated
4. FFmpeg merges chunks into final MP3

### Custom Post Type
- Podcasts are stored as `ai_podcast` post type
- Fully compatible with Elementor for custom layouts
- Auto-embeds modern audio player
- Shortcode available: `[ai_podcast_player]`

## Server Requirements

**Required:**
- PHP 7.4+
- WordPress 6.0+
- MySQL 5.7+

**Recommended:**
- FFmpeg (for merging audio chunks)
- If FFmpeg is not available, plugin will still work but each voice section will be separate files

**To install FFmpeg:**
```bash
# Ubuntu/Debian
sudo apt-get install ffmpeg

# CentOS/RHEL
sudo yum install ffmpeg

# macOS
brew install ffmpeg
```

## Troubleshooting

**"Generation stuck in processing"**
- Check Action Scheduler: Tools > Scheduled Actions
- Look for `aipg_process_generation` actions
- Check error logs in dashboard

**"FFmpeg not available"**
- Install FFmpeg on your server
- Or accept multiple audio files per podcast

**"API Error"**
- Verify API keys are correct
- Check you have credits/quota
- Review error logs in generation history

**"Timeout errors"**
- This shouldn't happen thanks to Action Scheduler
- If it does, check WordPress memory limit
- Increase: `define('WP_MEMORY_LIMIT', '256M');`

## Usage Tips

1. **Best Duration**: 10-15 minutes works best for engagement
2. **Voice Mapping**: Assign different genders for variety
3. **Intro/Outro**: Keep them under 50 words
4. **Languages**: Works best with English, Spanish, French, German
5. **Source Articles**: Longer articles (500+ words) make better podcasts
